# TWITTER COPY with nodejs
## Description
ceci est copy de l'application twitter faite en Nodejs

## Stack
- Nodejs avec le framework express
- PUG comme moteur de templates
- Mongodb pour la persistance des donnees
- Mongoose comme ORM
## Fonctionalites
- Creer un copmte utilisateur
- Ecrire des tweets
- Liker et partager des tweets
- Supprimer des tweets
